Newcastle United Foundation Website

Responsive Website Development & Ecommerce Booking System

The Challenge

Rentokil came to us requiring a solution that their Sales Teams could use wherever, whenever. They used a multitude of platforms and devices, so the requirement was to come up with a solution that worked both online and offline, across MS Windows, Android and iOS devices.

Our Solution

We started the whole design phase from scratch. Using their brand guidelines, we were allowed to develop something completely new based around their current corporate brand structure. The user journey was of paramount importance, as this was to be used globally, through every country Rentokil operates in. It had to be intuitive enough, scalable enough but also responsive enough to work on any device, from laptop down to an iPhone. The foundation of this application was based around a highly adaptable .Net framework, with natural multi-language support for any country. The multi-language element made the user interface and design process that even more important.

Once we had created and tested the user interface, all functional elements were to be handled via the .Net CMS. This gave the application great scalability and manageability. All elements are supported, maintained and configured through the CMS. A compiled MSI file is generated to support the stand-alone Windows application, plus native online versions for Android and iOS were built to support their field operatives tablets and handsets.

The application has been so well received, that Rentokil are already looking to commission a phase 2 application with even more functionality and support.
